## How to Add These Pages to WordPress

### Step 1: Create the Pages in WordPress Admin

1. Go to **WordPress Admin** → **Pages** → **Add New**
2. Create each page with these exact titles:
   - `Política de Privacidade`
   - `Termos e Condições`
   - `Trocas e Devoluções`
   - `Envio e Entrega`
   - `FAQ`

### Step 2: Set Page Templates

For each page, in the **Page Attributes** box (right sidebar):

1. Select **Template**: Choose the corresponding template:
   - `Política de Privacidade` → `Política de Privacidade`
   - `Termos e Condições` → `Termos e Condições`
   - `Trocas e Devoluções` → `Trocas e Devoluções`
   - `Envio e Entrega` → `Envio e Entrega`
   - `FAQ` → `FAQ - Perguntas Frequentes`

### Step 3: Set Page Slugs (URLs)

Set these exact slugs for each page:

- `politica-privacidade`
- `termos-condicoes`
- `trocas-devolucoes`
- `envio-entrega`
- `faq`

### Step 4: Publish the Pages

1. Click **Publish** for each page
2. The pages will automatically use the custom templates and styling
